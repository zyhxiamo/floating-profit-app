# 浮盈 app 项目说明

## 产品目标

制作一个苹果风的 Windows A 股桌面悬浮窗。折叠后只显示一只自选股票并轮动展示，空白区域透明，不遮挡桌面操作；展开后显示毛玻璃面板、自选股、大盘实时数据和显示设置。

## 当前版本

版本：`0.2.0`

已完成：

- A 股代码添加、自选股保存和轮动展示
- 公开行情源接入，东方财富优先，腾讯行情作为备用
- 上证指数、深证成指、创业板指实时展示
- 折叠与展开、桌面置顶、`Alt+Q` 快捷隐藏
- 字号、整体界面尺寸和涨跌颜色开关
- 在线反馈入口和 GitHub Issues 维护清单
- 官网、搜索引擎基础信息和自动发布流程
- Windows 绿色独立版自动打包流程

## 版本维护流程

1. 在 GitHub Issues 查看用户反馈。
2. 在本项目目录修改并测试。
3. 更新 `package.json` 和 `package-lock.json` 的版本号。
4. 提交并推送 `main`，官网会自动更新。
5. 创建并推送版本标签，例如 `v0.2.1`，GitHub 会自动生成 Windows 下载包。

## 项目目录

```text
src/                 Electron 桌面应用
docs/                官网
.github/workflows/   自动发布官网和下载包
.github/ISSUE_TEMPLATE/ 在线反馈表单
scripts/             本地打包脚本
release/             本地生成的绿色独立版，不提交到 Git
```
