# 浮盈 app 项目说明

## 产品目标

浮盈 app 是一个 Windows A 股桌面悬浮窗。折叠后只显示一只自选股票并轮动展示；展开后显示毛玻璃面板、自选股、大盘实时数据和显示设置。

## 当前版本

版本：`0.2.1`

已完成：

- A 股代码添加、自选股保存、折叠轮动、桌面置顶和 `Alt+Q` 快捷隐藏
- 东方财富行情优先、腾讯行情备用
- 字号、整体界面尺寸和涨跌颜色开关
- 软件内游客评价提交，本地草稿兜底
- 官网精选评价、累计下载次数和固定下载地址
- CloudBase 云函数、数据库接入、隐藏审核页和部署脚本
- GitHub Pages 与 Releases 作为备用镜像

## 目录

```text
src/                  Electron 桌面应用
docs/                 官网和隐藏审核页
cloudfunctions/api/   CloudBase HTTP 云函数
scripts/              打包、CloudBase 配置和部署脚本
release/              本地绿色独立版，不提交 Git
```

## 数据约定

- 游客评价只收集昵称和最多 100 字评价，不收集联系方式。
- 评价默认为 `pending`，审核通过后才会显示在官网。
- 下载接口每次跳转前累加累计下载次数。
- 正式版不包含打赏入口、收款二维码或个人支付信息。
