# 浮盈 app

浮盈 app 是一个 Windows A 股桌面悬浮行情工具，支持折叠轮动、自选股、真实行情、桌面置顶和快捷隐藏。

## 本地开发

```powershell
npm install
npm start
```

生成绿色独立版：

```powershell
.\生成绿色独立版.bat
```
