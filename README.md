# 浮盈 app

浮盈 app 是一个 Windows A 股桌面悬浮行情工具。它面向只想快速扫一眼自选股的用户，支持折叠轮动、展开面板、桌面置顶、快捷隐藏、真实行情和在线反馈。

## 下载
