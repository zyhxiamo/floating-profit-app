# 浮盈 app 介绍视频

## 成片

- `浮盈-app-抖音竖屏版.mp4`：1080 x 1920，约 26 秒。
- `浮盈-app-B站横屏版.mp4`：1920 x 1080，约 26 秒。
- `浮盈-app-抖音封面.png`：抖音竖屏封面。
- `浮盈-app-B站封面.png`：B 站横屏封面。

## 配套文件

- `浮盈-app-字幕.srt`：视频字幕。
- `浮盈-app-口播提示稿.md`：中文女声口播文本，也可作为真人口播参考。
- `generate-narration.ps1`：使用电脑自带中文女声生成口播音轨。
- `生成介绍视频.bat`：重新生成横屏和竖屏成片。

演示视频只使用示例股票，画面底部包含“行情仅作功能演示，不构成投资建议”提示。背景音乐为本地生成的轻快节拍，不依赖第三方音乐素材。
